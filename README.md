# Aspect Hebrew Landing Page

Static Hebrew website for GitHub Pages.

## Files

- `index.html` — main Hebrew landing page.
- `assets/Horizontal.png` — Aspect logo PNG supplied by the client.
- `assets/proforma-example.pdf` — PDF example shown and linked from the page.
- `assets/houston-skyline.jpg` — image extracted from the work document.
- `assets/neighborhood-street.jpg` — image extracted from the work document.
- `assets/homes-exterior.jpg` — image extracted from the work document.

## GitHub Pages upload

Upload `index.html` and the full `assets` folder to the GitHub repository root.
The page uses relative paths, so it will work from GitHub Pages without extra setup.
